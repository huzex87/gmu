# GMU-014 — Location Image Production Checklist

**Hand this document to whoever is generating location reference images.** Every prompt below is already fully assembled and ready to paste exactly as written — no editing, no lookups. Work top to bottom by location cluster; within each cluster, the order shown is the recommended generation order from that location's own brief. Check off each one as you lock it.

## Before You Start

Same accounts/tools/procedure as GMU-013 (Midjourney Standard, or FLUX). Same universal negative prompt applies to every generation:
> no anachronistic clothing or technology, no modern text or logos, no generic "fantasy Africa" tropes, no orientalist exoticization, no oversaturated or cartoonish color grading, no morphing or warping artifacts

## The Procedure (identical for every location below)

1. Copy the **PROMPT TO PASTE** block exactly as written.
2. Paste into Midjourney, add the universal negative prompt (`--no ...`), generate.
3. Pick the strongest of the 4 results, Upscale, Save.
4. Rename to the **EXACT FILENAME** given.
5. Save into the **EXACT FOLDER** given (all locations save into `19_CONCEPT_ART/`).
6. Check the box, move to the next location.

Locations generally don't need `--cref` locking the way characters do (no face/costume to hold steady), but if a location recurs across many chapters (e.g., the Palace's Sultan's Chamber, the Diwan Hall), it's still good practice to keep the accepted image's URL on hand in case you want to generate close-variant angles later with `--cref` for architectural consistency.
