# GMU-015 — Voice Casting Master Checklist

**Hand this document to whoever owns voice casting.** Every character's Voice Direction is already copied below, verbatim from their character sheet — no lookups needed. Do not start this until GMU-013 (character images) is fully checked off, since GMU-006's sequencing rule applies here too: cast every character's voice once, correctly, before generating any chapter's dialogue.

## Before You Start

1. Create an ElevenLabs account at elevenlabs.io.
2. Upgrade to the **Creator plan ($22/mo)** — Settings → Subscription. This is required; the free/Starter tiers only unlock Instant Voice Cloning, which degrades badly across a full season's dialogue for a recurring character.
3. Open `18_PROMPTS/GMU-007_Prompt_Library.md` — you will log the final voice ID for each character into Section 7 of that file once cast (a blank template row is provided per character below to fill in and then copy into GMU-007).

## The Procedure (identical for every character below)

1. Read the character's **Voice Direction** (copied below).
2. Go to ElevenLabs' **Voice Library** (left sidebar).
3. Filter by gender/age/accent to narrow candidates, then audition 3–5 stock voices using the built-in preview player — type a short test line and listen for a match to the direction notes below.
4. Found a match → click **Add to my voices**.
5. Have a real actor's sample + rights to use it? Go to **Voice Lab → Create Voice → Professional Voice Cloning (PVC)** instead, upload the required training audio, wait for training.
6. Fill in the **CASTING LOG** block below with the real Voice ID (found on the voice's detail page in ElevenLabs).
7. Check the box, move to the next character.

**PVC vs IVC:** use Professional Voice Cloning for all 13 named characters below (all speak across multiple chapters). Instant Voice Cloning is acceptable only for the one-scene background roles in `Background_and_Incidental_Characters.md` — those are not listed individually here since they don't need a locked voice ID.
